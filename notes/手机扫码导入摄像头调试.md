# 手机扫码导入摄像头调试

## 问题描述

手机端扫码导入功能没有任何反应，无法识别二维码。

## 问题原因

手机浏览器访问摄像头需要满足以下条件：
1. **HTTPS 环境**：现代浏览器要求 HTTPS 才能使用 `getUserMedia` API 访问摄像头
2. **摄像头权限**：用户需要授权摄像头访问权限
3. **视频流就绪**：需要等待视频流数据足够后再开始扫描

## 解决方案

### 1. 确保 HTTPS 环境
- 本地开发：使用自签名证书或 `localhost`
- 部署环境：必须使用 HTTPS

### 2. 添加详细调试日志
在关键位置添加日志以便诊断：

```javascript
// Camera.start()
console.log('[Camera] requesting camera access, facingMode:', facingMode);
console.log('[Camera] stream obtained, tracks:', stream.getTracks().length);
console.log('[Camera] video playing, videoWidth:', video().videoWidth);

// startQRScan()
console.log('[QR] Camera.start result:', ok);
console.log('[QR] video.readyState:', video.readyState);

// scanFrame()
if (frameCount % 30 === 0) {
  console.log('[Scanner] video state:', {
    readyState: video.readyState,
    videoWidth: video.videoWidth,
    videoHeight: video.videoHeight,
    paused: video.paused,
    srcObject: !!video.srcObject,
  });
}
```

### 3. 等待视频数据就绪
```javascript
const video = document.getElementById('video');
if (video.readyState >= video.HAVE_ENOUGH_DATA) {
  QRScanner.startScan(onQRScanResult);
} else {
  video.addEventListener('loadeddata', function onReady() {
    video.removeEventListener('loadeddata', onReady);
    if (qrScanMode) {
      QRScanner.startScan(onQRScanResult);
    }
  });
}
```

## 调试步骤

1. 打开手机浏览器开发者工具（Chrome Remote Debugging）
2. 触发扫码导入
3. 查看控制台日志，确认：
   - `Camera.start` 返回 `true`
   - `video.readyState` 达到 `HAVE_ENOUGH_DATA`（值为 4）
   - `videoWidth` 和 `videoHeight` 大于 0
