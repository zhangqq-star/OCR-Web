# 连续扫描（批量扫描）功能设计

## 概述

解决当前 v1.2 每次扫描后需手动选择位置的问题。新增"连续扫描"模式：预先设定起始位置和填充方向，每次 OCR 确认后自动写入下一格，实现批量扫描入库。

---

## 一、入口

### 1.1 扫描视图入口
- 在摄像头预览区下方新增按钮：`📦 连续扫描`
- 与现有一`📸 拍照识别`按钮的下方



---

## 二、起始位置选择弹窗

触发连续导入后，弹出设置弹窗，包含以下选项：

### 2.1 选择目标货架
- 下拉列表，列出所有货架
- 默认选中当前正在查看的货架

### 2.2 选择起始位置
- 展示 4×8 网格，与货架视图一致
- 已占用的格子显示为灰色/不可选（或显示为"将覆盖"提示）
- 空格子可点击选中，高亮标记为起始位置
- 默认选中第一个空格子

### 2.3 选择填充方向
- **先行后列**（默认）：从左到右依次填充第 0 行，再进入第 1 行
- **先列后行**：从上到下依次填充第 0 列，再进入第 1 列

### 2.4 预览填充路径
- 网格上以箭头或序号标注前 N 个填充位置
- 让用户直观看到零件将按什么顺序存放

### 2.5 覆盖策略
- **跳过已占用**：遇到有零件的位置自动跳过
- **覆盖已占用**（默认）：弹出确认，覆盖原有零件
- **遇占用停止**：到第一个已有零件的位置就停止

### 2.6 确认开始
- 点击"开始导入"按钮，关闭弹窗，进入连续扫描模式
- 显示当前进度状态栏

---

## 三、连续扫描流程

### 3.1 状态栏
屏幕顶部（或摄像头预览上方）常驻显示：

| 信息 | 说明 |
|------|------|
| 当前位置 | 如"第 2 行 第 5 列 (11/32)" |
| 进度条 | 已扫描数 / 可填充数 |
| 退出按钮 | 结束连续导入 |

### 3.2 扫描循环
每轮操作步骤（识别结果以全局弹窗形式浮在所有页面上方）：

```
┌────────────────────────────────────────┐
│          摄像头实时预览                  │
│  ┌ 状态栏：第 X 行 第 Y 列 · 已存 N 个 ┐ │
│  │                                    │ │
│  │          [摄像头画面]               │ │
│  │                                    │ │
│  └────────────────────────────────────┘ │
│  [📸 拍照识别]  [📦 连续导入]  ...      │
├────────────────────────────────────────┤
│  (背景半透明遮罩)                       │
│                                        │
│       ┌──────────────────────┐         │
│       │  📋 识别结果（弹窗）   │         │
│       │  零件编号：1234567890 │         │
│       │  目标：第 2 行 第 5 列 │         │
│       │                      │         │
│       │  [✅ 确认存入]        │         │
│       │  [⏭ 跳过] [🔄 重试]  │         │
│       │  [⌨ 手动] [✕ 结束]   │         │
│       └──────────────────────┘         │
│                                        │
└────────────────────────────────────────┘

流程循环：
  ① 摄像头实时预览 → ② 拍照 → ③ OCR 识别（spinner 覆盖摄像头）
  → ④ 全屏弹窗显示识别结果（背景半透明遮罩）
  → ⑤ 用户操作：
    [确认存入/跳过/手动输入] → 弹窗关闭，推进到下一位置，回到①
    [重试] → 弹窗关闭，回到②
    [结束] → 弹窗关闭，显示汇总弹窗
```

### 3.3 自动推进规则
- 确认存入或跳过 → 自动移动到下一个格子
- 根据"填充方向"和"覆盖策略"决定下一个目标位置
- 到达货架末尾（无下一个可用位置）→ 自动结束，提示"货架已满"

### 3.4 快速操作（可选增强）
- 引入倒计时自动确认：识别成功 3 秒后自动存入并进入下一次拍照
- 开关选项，用户可在设置中开启/关闭

---

## 四、结束汇总

退出连续导入模式后，弹出汇总卡片：

| 汇总项 | 说明 |
|--------|------|
| 导入成功 | N 个零件已存入 |
| 跳过 | N 个位置留空 |
| 覆盖 | N 个位置被替换 |
| 耗时时长 | 本次连续导入用时 |

- 点击"查看货架"跳转到货架视图
- 点击"继续导入"回到起始位置选择

---

## 五、异常处理

| 场景 | 处理方式 |
|------|---------|
| 摄像头权限丢失 | 暂停流程，提示重新授权 |
| OCR 识别失败（全模式无结果） | 提示"未识别到编号，请重试或手动输入" |
| 所有位置已满 | 自动结束，提示"货架已无空位" |
| 页面刷新/关闭 | 下一次进入时状态栏提示"上次有未完成的连续导入"，可恢复 |
| 连续导入中途切换到货架 Tab | 继续保留连续导入状态，切回扫描 Tab 可继续 |

## 七、与现有功能的交互

- **与手动添加兼容**：连续导入跳到空格，用户也可中途手动在货架视图添加零件
- **与长按移动兼容**：连续导入期间不能在货架视图做长按移动（避免位置冲突）
- **与 Excel 导出兼容**：连续导入结束后，导出功能正常使用
