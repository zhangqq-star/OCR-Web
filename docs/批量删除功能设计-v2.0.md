# 批量删除功能设计

## 入口
货架页面顶部工具栏新增"批量管理"按钮，点击进入批量管理模式。
（长按移动/交换行为保持不变，互不干扰。）

## 批量管理模式
- 所有已占用格子左上角显示圆形复选框，默认未选中
- 点击已占用格子切换选中/取消
- 点击空格子无效
- 选中格子高亮

## 底部操作栏
进入模式后底部浮现操作栏：
- **全选** / **取消全选**
- **删除选中 (N)** — N 为选中数量，0 时禁用
- **退出** — 恢复正常模式

## 删除流程
1. 点击"删除选中" → "确定要删除选中的 N 个零件吗？"
2. 确认 → 逐个调用 DataStore.removePart()
3. 退出批量模式，刷新货架，Toast "已删除 N 个零件"

## 技术要点
- shelf.js 新增 batchDeleteMode 状态
- 工具栏新增按钮，绑定进入/退出模式的事件
- 底部操作栏仅在批量模式下显示
- 删除用 for 循环逐个删，避免 SQLite 并发写冲突
