# 货架管理系统 v1.2 — 功能简述

## 概述

本地化 OCR 货架管理系统，基于 PWA 技术栈，支持摄像头拍照识别零件编号、货架网格可视化管理、多货架切换、Excel 导出等功能。纯前端运行，数据存储在浏览器 SQLite（SQL.js + OPFS）中，无需后端服务器。

## 技术栈

| 模块 | 技术 |
|------|------|
| OCR 引擎 | Tesseract.js v5（中英文混合识别） |
| 数据存储 | SQLite（SQL.js WASM + OPFS 持久化） |
| Excel 导出 | SheetJS (xlsx) v0.18 |
| 离线支持 | Service Worker + Web App Manifest |
| 图像预处理 | Canvas API（灰度化、高斯模糊、Sauvola 自适应二值化、形态学去噪） |

## 功能模块

### 1. 摄像头拍照与 OCR 识别

- 调用设备摄像头实时预览
- 拍照后自动进行图像预处理（针对黄色/红色背景优化，自适应通道权重）
- 预处理流程：灰度化 → 高斯模糊 → Sauvola 自适应二值化 → 中值去噪
- 多模式识别策略：
  - **常规模式**：中英文混合识别，从结果中提取 10 位数字编号
  - **数字白名单模式**：仅识别 0-9，针对纯数字标签优化
  - **原图模式**：预处理无效时回退使用原始图片
- 自动修正常见 OCR 混淆（如 O→0、I→1、S→5、Z→2、B→8）
- 识别结果实时展示，支持确认存入货架或重新识别

### 2. 货架网格管理

- 固定 **4 行 × 8 列** 网格布局，共 32 个库位
- 每个格子显示：位置号、零件编号（10 位完整显示）、名称、数量
- 点击空格子可手动添加零件
- 点击已占用格子查看/编辑零件详情
- 已占用格子高亮显示，空位显示 "+" 号
- 位置选择弹窗：OCR 识别后可视化选择存放位置，已占用位置灰色不可选

### 3. 多货架支持

- 支持创建、重命名、删除多个货架
- 顶部导航栏左右箭头切换货架
- 触摸滑动切换货架（导航栏区域）
- 货架切换带过渡动画
- 当前货架序号显示（如 "2 / 5"）
- 删除货架时关联零件一并删除（至少保留一个货架）

### 4. 零件 CRUD

- **添加**：手动输入或 OCR 识别后存入，必填 10 位数字编号
- **查看/编辑**：点击格子弹出详情弹窗，可修改编号、名称、规格、数量、备注
- **删除**：详情弹窗内删除按钮，需确认
- **移动**：长按格子（500ms）进入移动模式，点击目标位置完成移动
- **交换**：长按后点击已有零件的格子，两个零件互换位置

### 5. Excel 导出

- 一键导出当前货架数据为 .xlsx 文件
- 支持导出全部货架数据，每个货架一个 Sheet
- 导出字段：序号、编号、名称、规格、数量、货架行、货架列、备注
- 数据按货架行升序、货架列升序排列，与网格视觉位置一致
- 文件名自动带日期（如 `货架清单_2026-05-24.xlsx`、`全部货架清单_2026-05-24.xlsx`）

### 6. PWA 离线支持

- Service Worker 缓存静态资源，支持离线访问
- Web App Manifest 配置，可添加到手机主屏幕
- 独立图标和启动画面

### 7. UI/UX

- 毛玻璃风格（Glassmorphism）深色主题
- 流动背景光斑动画
- 底部 Tab 切换（扫描 / 货架）
- Toast 消息提示
- 窗口大小自适应，移动端优化
